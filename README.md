---

## 📌 Sobre o projeto

Este projeto foi desenvolvido como parte da **Atividade Extensionista II** do curso de Análise e Desenvolvimento de Sistemas (ADS) da UNINTER, em parceria com a **Associação de Moradores do Bairro Areia Branca**, em Petrolina/PE.

O objetivo é promover a **inclusão digital** de adultos com dificuldades na escrita formal, utilizando a Inteligência Artificial como ferramenta de apoio — não como substituta do pensamento humano.

---

## 🎯 Objetivos de Desenvolvimento Sustentável

- **ODS 04** — Educação de qualidade
- **ODS 10** — Redução das desigualdades

---

## 📦 Materiais desenvolvidos

### 1. Aplicação web — Gerador de Pedidos Formais

Ferramenta web desenvolvida em **HTML, CSS e JavaScript** que permite ao usuário:

- Escolher o tipo de pedido (requerimento, carta de benefício, mensagem para escola, solicitação à associação ou currículo)
- Preencher campos simples com suas informações
- Gerar automaticamente um texto formal pronto para copiar, usar ou enviar ao ChatGPT

🔗 **Acesse a aplicação:** [https://SEU-USUARIO.github.io/gerador-pedidos-formais](https://SEU-USUARIO.github.io/gerador-pedidos-formais)

### 2. Guia educativo em PDF

Guia de 4 páginas em linguagem acessível contendo:

- O que é a Inteligência Artificial
- Passo a passo para acessar o ChatGPT gratuitamente
- 5 modelos prontos de pedidos formais
- Orientações de uso responsável e ético
- Glossário de termos digitais

📄 O arquivo `guia_uso_responsavel_ia.pdf` está disponível neste repositório.

## 🚀 Como usar a aplicação

1. Acesse o link da aplicação pelo celular ou computador
2. Escolha o tipo de pedido que precisa escrever
3. Preencha os campos com suas informações
4. Clique em **Gerar texto**
5. Copie o texto gerado e use como quiser — envie ao ChatGPT para melhorar ou cole direto onde precisar
